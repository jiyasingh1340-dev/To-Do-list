# 📝 Todo List

A simple and attractive Todo List application built using **HTML, CSS, and JavaScript**.

## 🚀 Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Toggle task status (Completed / Pending)
- Set task priority (High, Medium, Low)
- Display tasks using dummy data
- Task counter (Total, Completed, Pending)
- Responsive and attractive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
